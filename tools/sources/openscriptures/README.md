# OpenScriptures lexicon sources

These files are bundled so the generated offline lexicons can be rebuilt without
depending on temporary downloads.

- `strongs-hebrew-dictionary.js` from https://github.com/openscriptures/strongs
- `strongs-greek-dictionary.js` from https://github.com/openscriptures/strongs
- `LexicalIndex.xml` from https://github.com/openscriptures/HebrewLexicon

The Strong dictionary JavaScript files retain their original copyright and
license notices. The OpenScriptures Hebrew Lexicon project releases its project
files under CC BY 4.0 and states that the BDB and Strong dictionary texts are in
the public domain. Credit: Open Scriptures Hebrew Bible Project.
